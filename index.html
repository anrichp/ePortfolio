Object-oriented Information Systems

The module examines the different elements of an information system from design through to processing and storage. You will explore the fundamental concepts behind object-oriented development using UML as a tool for analysis and design. Further to this, we will make use of Python to provide concrete experience of object-oriented development.

Network and Information Security Management

This module will give you a grounding in security concepts and terminology, illustrated by hands on practice in one of the most important aspects of information security today – network technology. We do not assume any prior knowledge of either topic, so we will start at the grass roots level, providing knowledge in the basics of both information security and network technology. We will use the latter to illustrate some very important security concepts such as vulnerability assessments, penetration testing, forensic analysis and breach management. We will conclude the module with a look at some of the challenges that face the Internet as we know it today, and consideration of how recent and emerging network technology can be used to mitigate some of the issues presented.

Secure Software Development

This module will take an in depth look at the security risks that are associated with programming languages, from the perspectives of design and architecture approaches, programming paradigms, testing, the impact and implications of operating systems and libraries on software development, the security risks associated with distributed systems and APIs, and future trends in secure software and systems design. Running through these topics will be the concept of the secure software development life cycle.
